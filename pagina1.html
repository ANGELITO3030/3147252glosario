Glosario A-M

Glosario de Tecnología de la Información

(A-M)

A B C D E F G H I J K L M N-Z →

A

Algoritmo: Conjunto de instrucciones o reglas definidas para resolver un problema o realizar una tarea.

API: Interfaz de programación de aplicaciones que permite la comunicación entre diferentes sistemas.

Autenticación: Proceso de verificar la identidad de un usuario o sistema.

ASCII: Código estándar para la representación de caracteres en computadoras.

Arquitectura cliente-servidor: Modelo de red donde los clientes solicitan servicios a un servidor.

B

Base de datos: Conjunto de datos organizados que se almacenan y se pueden acceder de manera eficiente.

Big Data: Conjunto de datos masivos que requieren tecnologías avanzadas para su procesamiento.

Backup: Copia de seguridad de datos para prevenir pérdidas.

BIOS: Sistema básico de entrada y salida que inicializa el hardware de una computadora.

Blockchain: Tecnología de registro distribuido utilizada para garantizar la seguridad y transparencia.

C

Cloud Computing: Modelo de computación que permite el acceso a recursos informáticos a través de Internet.

CPU: Unidad central de procesamiento que ejecuta instrucciones en un sistema informático.

Cache: Memoria de acceso rápido utilizada para almacenar datos frecuentemente utilizados.

Criptografía: Técnica para proteger información mediante el cifrado.

Compilador: Programa que traduce código fuente a código máquina.

D

Desarrollo ágil: Metodología de desarrollo de software que promueve la entrega rápida y continua de productos.

DNS: Sistema de nombres de dominio que traduce nombres de dominio a direcciones IP.

Data Mining: Proceso de analizar grandes conjuntos de datos para encontrar patrones.

Driver: Software que permite la comunicación entre el sistema operativo y el hardware.

Debugging: Proceso de identificar y corregir errores en el código.

E

Encriptación: Proceso de convertir información en un código para prevenir el acceso no autorizado.

Ethernet: Tecnología de red utilizada para conectar dispositivos en una red local.

Edge Computing: Procesamiento de datos cerca de la fuente de generación para reducir la latencia.

Entorno de desarrollo: Conjunto de herramientas utilizadas para programar y probar software.

Escalabilidad: Capacidad de un sistema para manejar un aumento en la carga de trabajo.

F

Firewall: Sistema de seguridad que controla el tráfico de red y protege contra accesos no autorizados.

Firmware: Software integrado en hardware que controla su funcionamiento.

FTP: Protocolo de transferencia de archivos utilizado para transferir datos entre dispositivos.

Framework: Conjunto de herramientas y bibliotecas para desarrollar aplicaciones.

Función hash: Algoritmo que convierte datos en una cadena de longitud fija.

G

GPU: Unidad de procesamiento gráfico, especializada en el manejo de gráficos y procesamiento paralelo.

Git: Sistema de control de versiones distribuido utilizado para gestionar proyectos de software.

Gateway: Dispositivo que conecta diferentes redes y permite la comunicación entre ellas.

Gestión de proyectos: Proceso de planificar, ejecutar y supervisar proyectos.

Gigabyte: Unidad de medida de almacenamiento equivalente a 1,024 megabytes.

H

Hardware: Conjunto de componentes físicos que forman un sistema informático.

HTML: Lenguaje de marcado utilizado para estructurar contenido en la web.

HTTP: Protocolo de transferencia de hipertexto utilizado para la comunicación en la web.

Hosting: Servicio que permite alojar sitios web en servidores.

Hipertexto: Texto que contiene enlaces a otros documentos o recursos.

I

Inteligencia Artificial: Rama de la informática que se ocupa de crear sistemas capaces de realizar tareas que requieren inteligencia humana.

IP: Protocolo de Internet utilizado para identificar dispositivos en una red.

IDE: Entorno de desarrollo integrado que facilita la programación.

IoT: Internet de las cosas, red de dispositivos conectados que interactúan entre sí.

Instancia: Ejecución de un objeto o clase en programación orientada a objetos.

J

Java: Lenguaje de programación orientado a objetos, ampliamente utilizado en el desarrollo de aplicaciones.

JavaScript: Lenguaje de programación utilizado para crear contenido interactivo en la web.

JSON: Formato de intercambio de datos ligero y legible por humanos.

JVM: Máquina virtual de Java que ejecuta programas escritos en Java.

Jupyter Notebook: Herramienta interactiva para escribir y ejecutar código en múltiples lenguajes.

K

Kernel: Núcleo del sistema operativo que gestiona los recursos del hardware y permite la comunicación entre software y hardware.

Kubernetes: Plataforma de código abierto para la gestión de contenedores.

Keylogger: Software malicioso que registra las pulsaciones del teclado.

Key-value store: Tipo de base de datos que almacena datos como pares clave-valor.

Knapsack problem: Problema de optimización combinatoria en matemáticas y computación.

L

Lenguaje de programación: Conjunto de reglas y símbolos utilizados para crear programas informáticos.

Linux: Sistema operativo de código abierto basado en Unix.

Latencia: Tiempo que tarda un paquete de datos en viajar de un punto a otro.

Load Balancer: Dispositivo o software que distribuye el tráfico de red entre varios servidores.

Licencia de software: Contrato que define cómo se puede usar un software.

M

Machine Learning: Subcampo de la inteligencia artificial que permite a las máquinas aprender de datos y mejorar su rendimiento.

Memoria caché: Memoria de acceso rápido utilizada para almacenar datos frecuentemente utilizados.

Microprocesador: Circuito integrado que contiene la unidad central de procesamiento.

Middleware: Software que actúa como intermediario entre diferentes aplicaciones o sistemas.

Metadatos: Datos que describen otros datos, como su contenido o estructura.
