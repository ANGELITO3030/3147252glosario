Glosario de Tecnología de la Información

(N-Z)

← A-M N O P Q R S T U V W X Y Z

N

Networking: Conexión de computadoras y dispositivos para compartir recursos e información.

Node.js: Entorno de ejecución para JavaScript que permite crear aplicaciones del lado del servidor.

Nanosegundo: Unidad de tiempo equivalente a una milmillonésima parte de un segundo.

Nombre de dominio: Identificador único que se utiliza para acceder a sitios web en Internet.

NIC (Network Interface Card): Tarjeta de red que permite la conexión de un dispositivo a una red.

O

Open Source: Software cuyo código fuente es accesible y puede ser modificado y distribuido por cualquier persona.

Operador lógico: Símbolo utilizado en programación para realizar operaciones lógicas como AND, OR y NOT.

OCR (Reconocimiento Óptico de Caracteres): Tecnología que convierte texto en imágenes a texto editable.

Overclocking: Técnica para aumentar la velocidad de un procesador más allá de sus especificaciones.

Ontología: Representación formal de un conjunto de conceptos y sus relaciones en un dominio específico.

P

Protocolos: Conjunto de reglas que determinan cómo se comunican los dispositivos en una red.

Python: Lenguaje de programación de alto nivel conocido por su simplicidad y versatilidad.

PHP: Lenguaje de programación utilizado principalmente para el desarrollo web del lado del servidor.

Ping: Herramienta de red utilizada para comprobar la conectividad entre dispositivos.

Procesador: Unidad central de procesamiento que ejecuta instrucciones en un sistema informático.

Q

QoS (Quality of Service): Mecanismo que garantiza un nivel de rendimiento en la transmisión de datos en redes.

Query: Consulta utilizada para recuperar información de una base de datos.

Quantum Computing: Tecnología de computación basada en principios de la mecánica cuántica.

QuickSort: Algoritmo de ordenamiento eficiente basado en el método de dividir y conquistar.

Quark: Unidad fundamental de la materia en física de partículas.

R

Redes: Conjunto de dispositivos interconectados que pueden comunicarse entre sí.

Router: Dispositivo que dirige el tráfico de datos entre redes diferentes.

RAM (Memoria de Acceso Aleatorio): Memoria volátil utilizada para almacenar datos temporales.

Repositorio: Almacenamiento centralizado para gestionar y compartir código fuente.

Resolución: Número de píxeles que componen una imagen en una pantalla.

S

Seguridad informática: Prácticas y tecnologías utilizadas para proteger sistemas y datos de accesos no autorizados.

Servidor: Computadora o programa que proporciona servicios a otros dispositivos en una red.

Script: Archivo que contiene un conjunto de instrucciones ejecutables por un programa.

SQL: Lenguaje utilizado para gestionar y consultar bases de datos relacionales.

SSL (Secure Sockets Layer): Protocolo de seguridad para cifrar la comunicación en Internet.

T

TCP/IP: Protocolo de comunicación que permite la transmisión de datos en Internet.

Token: Unidad de datos utilizada en la autenticación y autorización en sistemas informáticos.

Thread: Hilo de ejecución dentro de un proceso en un sistema operativo.

Terabyte: Unidad de medida de almacenamiento equivalente a 1,024 gigabytes.

Trojan: Tipo de malware que se disfraza como software legítimo.

U

UI (Interfaz de Usuario): Conjunto de elementos visuales que permiten la interacción entre el usuario y el software.

URL: Dirección única que identifica un recurso en Internet.

UDP: Protocolo de comunicación que permite la transmisión de datos sin establecer una conexión previa.

Unicode: Estándar de codificación que permite representar texto en diferentes idiomas y sistemas.

UML: Lenguaje de modelado unificado utilizado para diseñar sistemas de software.

V

Virtualización: Tecnología que permite crear versiones virtuales de recursos físicos, como servidores o sistemas operativos.

VPN: Red privada virtual que permite conexiones seguras a través de redes públicas.

Variable: Espacio de memoria utilizado para almacenar datos en un programa.

Versionado: Proceso de gestionar diferentes versiones de un archivo o software.

VLAN: Red de área local virtual que segmenta redes físicas en redes lógicas.

W

Web: Conjunto de páginas y recursos accesibles a través de Internet.

Wi-Fi: Tecnología que permite la conexión inalámbrica a redes de área local.

Widget: Elemento gráfico interactivo en una interfaz de usuario.

WordPress: Sistema de gestión de contenido utilizado para crear sitios web.

WAN: Red de área amplia que conecta redes locales en diferentes ubicaciones geográficas.

X

XML: Lenguaje de marcado que define reglas para la codificación de documentos en formato legible por humanos y máquinas.

XSS (Cross-Site Scripting): Vulnerabilidad de seguridad en aplicaciones web.

XOR: Operador lógico exclusivo utilizado en programación.

Xamarin: Plataforma para desarrollar aplicaciones móviles multiplataforma.

Xen: Hipervisor utilizado para la virtualización de sistemas operativos.

Y

Yottabyte: Unidad de medida de almacenamiento de datos equivalente a un billón de gigabytes.

YAML: Lenguaje de serialización de datos legible por humanos.

Yield: Palabra clave en algunos lenguajes de programación para generar valores en iteradores.

Yahoo: Empresa pionera en servicios de Internet y motores de búsqueda.

YUV: Formato de codificación de color utilizado en procesamiento de video.

Z

Zona desmilitarizada: Área de red que se encuentra entre una red interna y una red externa, utilizada para aumentar la seguridad.

ZIP: Formato de compresión de archivos ampliamente utilizado.

Zettabyte: Unidad de medida de almacenamiento de datos equivalente a 1,024 exabytes.

Zigbee: Protocolo de comunicación inalámbrica para dispositivos de bajo consumo.

Z-buffer: Técnica utilizada en gráficos 3D para gestionar la profundidad de los píxeles.
